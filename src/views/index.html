Usage

Get a link info

Making request:
{{.scheme}}://{{.host}}:{{.port}}/api?link=link_to_short
It will return a json reponse where:
  • link is your transfered link
  • hash is unique hash of link
  • transitions is ammount of made transition on shorted link
  • address is shorted link
  • is_new it means is link was created now at first time
